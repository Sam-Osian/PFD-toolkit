1. Who We Are

This website is operated as PFD Toolkit. This policy explains how we process personal data when you use the site.

2. What Data We Process

Depending on how you use the site, we may process:

  • Technical request data such as IP address, browser details, and timestamps (for security and operations).
  • Session and anti-forgery identifiers used to keep the app working securely.
  • Workspace content you create in the app (for example workbook names and report selections).
  • Configuration values you enter in the app (for example model and provider settings).

3. Cookie Notice (Cookies And Similar Technologies)

We use cookies and browser storage to run the service securely and remember interface choices. We currently do not use advertising cookies and do not currently load third-party analytics tags.

Technology Name Type First/third party Purpose Typical duration
Cookie csrftoken Strictly necessary First party Prevents cross-site request forgery attacks. About 1 year
Cookie sessionid Strictly necessary First party Keeps your session state between requests. Until session expiry or browser close
Local storage workbench.* (for example workbench.sidebarCollapsed) Preferences First party Stores UI preferences such as sidebar and panel expansion state. Until you clear browser storage

We do not currently set known non-essential third-party analytics or advertising cookies through this site. If that changes, we will update this notice and implement consent controls where required.

4. Managing Cookies And Storage

You can manage or delete cookies and local storage using your browser settings. Blocking strictly necessary cookies may affect security or break parts of the service.

For questions about this cookie notice, contact hello@pfdtoolkit.org.
